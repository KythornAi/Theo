# Theo — Brain Vault Output

This folder receives research reports and analysis written by Theo (Hermes Agent on Pi) when he uses the `obsidian-write` skill. Files here are finished, processed outputs — not raw research-in-progress.

## What belongs here

- Competitive analysis writeups
- Market research summaries
- Processed research reports flagged for vault storage

Raw research-in-progress lives in `Side_Hustle/theo/research/*.md` (see index below).

## File naming convention

All files are prefixed `theo-` so they're clearly agent-authored:

```
theo-<topic-slug>-YYYY-MM-DD.md
```

## How Theo writes here

Theo uses the `obsidian-write` skill, which writes to `~/hermes_files/theo/brain_out/` on the Pi. Syncthing syncs that folder to this vault directory automatically (usually within 30 seconds). Theo also flags each write in `HERMES_INBOX.md` with `[BRAIN-WRITE]`.
