## Summary & Key Insights

### Price Ranges
- **Premium tier:** $17.28 (FutureADHD) — highest reviews, most features
- **Mid tier:** $11-12 (DailyFocusClub, DanWLifePlannerStore) — science-based, many templates
- **Budget tier:** $0.99-$6.99 (Manifestable, PlanRightCo, LetzPlan) — aggressive discounting
- **Ultra-budget:** $1.19-$2.25 (LetzPlan, KlassicCreatives) — massive page/sticker counts as value props

### Most Common Features Across Top Sellers
- **Hyperlinked navigation** (18/20 listings)
- **GoodNotes compatibility** (18/20)
- **Monday & Sunday start options** (15/20)
- **Brain dump templates** (12/20)
- **Habit/mood tracking** (14/20)
- **Finance/budget trackers** (12/20)
- **Digital stickers included** (14/20)
- **Multiple covers/themes** (10/20)
- **Dark mode variants** (8/20)
- **Undated options** (8/20, often alongside dated)
- **Calendar integration** (Apple/Google Calendar) (5/20)

### Unique ADHD-Specific Features (Differentiators)
- **FutureADHD**: Hyperfocus Lotus, Impulse Buy Checklist, 10&10 Savings Method, Chore Chopper, "shame spiral" prevention
- **DailyFocusClub**: Apple Reminders/Google Calendar shortcuts, GTD methodology
- **BrainWaveFocusShop**: Dopamine tracking, low-stimulation design, Domestic Bliss hub
- **PlanRightCo**: Rejection sensitivity tools, inner critic/compassion features, analysis paralysis beaters

### Market Gaps / Opportunities
1. **No dominant spreadsheet-based ADHD planner** — mostly PDF/Goodnotes format
2. **Calendar auto-sync is rare** — most offer tap-to-open links only
3. **Notion templates for ADHD are underrepresented** in Etsy search despite Reddit demand
4. **Printable format has few truly ADHD-specific options** — most are just standard planners with ADHD label
5. **Academic/school-year (July-June) format** — mentioned as high demand in customer feedback but underserved
6. **Interactive HTML products** — none of the top sellers offer web-based interactive planners, all are PDF downloads
