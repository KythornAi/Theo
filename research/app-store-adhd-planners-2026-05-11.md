## User Love / Hate Signals

### What Users LOVE (Across All Apps)

1. **Task breakdown into smaller steps** — mentioned most consistently. neurolist users rave about AI breaking tasks down. Splitti users love the "hidden smaller tasks" discovery
2. **Focus timer / Pomodoro** — consistently praised. Seen as essential, not optional
3. **Reminders & notifications** — "just the right timing" is the differentiator. Too early = ignored; too late = useless
4. **Saved task libraries / templates** — being able to reuse lists is repeatedly called out as valuable
5. **Simplicity / minimal UI** — "not too many options" wins over "comprehensive feature set"
6. **Visual timeline / time representation** — Tempus's visual full-day timeline specifically praised for combating time blindness

### What Users HATE / Complain About

1. **Hidden subscription costs / lack of pricing transparency** — Goblin Tools, Splitti, and Easy-Do all got nailed for this. Users HATE finding out mid-trial
2. **Paywalls after minimal use** — neurolist users angry they hit a paywall after 5 tasks. "Blindsided by a subscription"
3. **Data loss / sync failures** — Goblin Tools users reported tasks disappearing from calendar. Trust killer
4. **Registration/login friction** — Splitti users complained about having to "fill this stupid thing out every time"
5. **Can't edit AI-generated task durations** — Super Planner users couldn't adjust times AI set
6. **Missing features:** stopwatch (Easy-Do), ability to set specific times for appointments (Unique), home screen widgets (Planwoo), Apple Watch app (Tempus)
7. **Free trial requires credit card** — Unique users flagged this as frustrating

## Key Patterns & Opportunities for PNP

### 1. Task Breakdown is the Core Value Proposition
Every successful app offers task breakdown — either AI-automated or user-driven. **Our app already does this (user-driven).** The gap: we don't have a **saved task library** so users can reuse broken-down lists. neurolist's "Task Library" is specifically praised. A library of pre-broken-down common ADHD tasks ("clean the kitchen", "start a project", "plan a trip") would give immediate value without needing AI.

### 2. Saved Lists / Templates is a Hunger Signal
Users repeatedly mention wanting to save and return to task lists. This is low-hanging fruit — a template system where users can "save this plan and come back to it" directly addresses the abandonment cycle (they can pick up where they left off without re-planning).

### 3. Focus Timer + Reminders = Table Stakes
These aren't differentiators — they're expectations. Every planner app has a focus timer and reminders. **Our planner needs both to be taken seriously.** Even a simple countdown timer with a notification would match the baseline.

### 4. Pricing Transparency is a Competitive Weapon
Almost every app store review mentions confusion or anger about pricing. **Being upfront** ("one price, no subscription, this is what you get") could be a significant trust advantage against the $5-15/month subscription apps.

### 5. The "By an ADHDer" Frame is Underused in Apps
FutureADHD and Gravity Focus both leverage "created by someone with ADHD." Most app store planners don't lead with this. If PNP's planner uses ADHD-friendly design language AND positions itself as "designed by and for ADHD brains," it stands out in both the app and Etsy markets.

### 6. Voice Input is a Niche Opportunity
Super Planner's voice capture ("ADHD minds think faster than they type") is only offered by one app in our scan. If our web planner supported speech-to-text for task entry, that's a genuine differentiator that directly addresses the "capture speed" pain point from the buyer persona.

## Market Gaps in App Store (vs Etsy)

Where app store planners consistently fail where an HTML/PDF planner could win:

| App Store Gap | How It Manifests | Our Advantage |
|---------------|------------------|---------------|
| Subscription fatigue | $5-15/mo per app. Users manage multiple subscriptions | One-time purchase price (~£12-15). No recurring cost |
| Setup friction | Login, credit card, tutorial, configuration | Open URL, start planning. Zero friction |
| Platform lock-in | Android/iOS only. Can't switch devices | Any browser. Phone, tablet, laptop, same data |
| Data portability | Can't export, can't print | One-click print / export to PDF |
| Abandonment guilt | Streak counters, "you missed 3 days" notifications | Welcome back, no guilt. Fresh start always |
| AI unreliability | AI tasks disappear or wrong durations | User-controlled breakdown. No black box |
