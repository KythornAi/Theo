# Pin Strategy — Paper N Pixels

> Per content pillar, 3 pin formats, using hooks verified by Pinterest best practices.
> All pins: 1000×1500px (2:3 ratio), text overlay, mobile-readable font.

---

## Pin Format Templates

### Template A: Problem → Empathy
| Element | Content |
|---------|---------|
| Image | Warm, soft background (PNP brand: #DFD3B8), simple illustration or handwritten-style text |
| Text overlay | Large: "Your brain isn't broken" / "Your planner is just built for a different brain" |
| Pin title | [Keyword] — why [problem] happens to ADHD brains |
| Description | 2-3 sentences naming the pain + offering the solution. Links to blog post. |

### Template B: Before/After
| Element | Content |
|---------|---------|
| Image | Split screen or two-panel. Left: messy/stressed. Right: calm/simple |
| Text overlay | Left: "Before: 15 undone tasks" / Right: "After: 1 intention, done" |
| Pin title | What [problem] looks like vs what it CAN look like with ADHD |
| Description | "Here's what changed when I stopped trying to plan like a neurotypical person." Links to blog. |

### Template C: Step Guide
| Element | Content |
|---------|---------|
| Image | Numbered 1-2-3 steps with simple icons. Clean, minimal |
| Text overlay | "3 steps to reset an overwhelmed ADHD brain" |
| Pin title | [Number] simple steps to [solve problem] when you have ADHD |
| Description | "It doesn't have to be complicated. Here's the exact process." Links to blog. |

## Pin Hook Library (by pillar)

### Task Initiation Hooks
- "One thing. That's all you need to plan today."
- "Your to-do list is the problem. Let me explain why."
- "Task paralysis isn't laziness. Here's what's actually happening."
- "The 'start anywhere' method: how I stop staring at my list"

### Brain Dump Hooks
- "Your brain is holding 15 thoughts right now. Here's how to release them."
- "Empty your head in 60 seconds. No structure. No rules. Just relief."
- "The brain dump is not optional. It's how ADHD brains unclog."
- "Overwhelmed? Do this one thing before anything else."

### Time Blindness Hooks
- "Hours disappear. It's not your fault."
- "Time blindness isn't a character flaw. It's a brain wiring thing."
- "Why 'check the clock more' doesn't work for ADHD"
- "The 10-minute anchor: one tool that saved my days from disappearing"

### Planner Abandonment Hooks
- "4 planners abandoned. What changed? The fifth one wasn't a planner."
- "Streaks are shame with a number attached."
- "You don't need more discipline. You need a different tool."
- "The undated planner solved my planner abandonment cycle"

### Low Energy Hooks
- "Some days, the goal is survival. Here's a plan for those days."
- "Crash day checklist: what matters when nothing works"
- "Your planner should have a low-energy mode. Most don't."
- "The bare minimum is enough. Say it again."
